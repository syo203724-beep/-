## 音声生成ツール

### 方法1: VOICEVOX（推奨・無料）

1. **ダウンロード**: https://voicevox.hiroshiba.jp/
2. **起動**: VOICEVOXを開く
3. **テキスト入力**: 上記のナレーションをコピペ
4. **キャラ選択**: 好みの声を選択（四国めたん、ずんだもん等）
5. **調整**: 速度を0.9〜1.0、抑揚を調整
6. **出力**: WAVで書き出し → MP3に変換

### 方法2: RVC（音声変換）

既存の音声をRVCで変換して自然な声にする場合:

1. **VOICEVOX等で基本音声を生成**
2. **RVC WebUI**: https://github.com/RVC-Project/Retrieval-based-Voice-Conversion-WebUI
3. **モデル選択**: 好みの声モデルを用意
4. **変換**: 基本音声を入力 → 変換後の音声を出力
5. **MP3で保存**

### 方法3: オンラインTTS

- **Google Cloud TTS**: https://cloud.google.com/text-to-speech
- **Amazon Polly**: https://aws.amazon.com/polly/
- **ElevenLabs**: https://elevenlabs.io/

## ファイル変換

WAV → MP3 変換が必要な場合:

```bash
# FFmpegを使用
ffmpeg -i input.wav -codec:a libmp3lame -qscale:a 2 output.mp3

# 全ファイルを一括変換
for f in *.wav; do ffmpeg -i "$f" -codec:a libmp3lame -qscale:a 2 "${f%.wav}.mp3"; done
```

---

## 推奨設定

- **サンプルレート**: 44100Hz または 48000Hz
- **ビットレート**: 128kbps 以上
- **チャンネル**: モノラルまたはステレオ
- **速度**: ゆっくり目（0.9〜1.0倍速）
- **抑揚**: やや強め（プレゼンテーション向け）
